# Timewise

A time-blocking to-do app built for the HCI course (Prof. Emanuele Panizzi, Sapienza Università di Roma, 2025-2026).

Unlike traditional to-do lists, Timewise treats estimated time as the primary planning unit — tasks are placed as blocks on a daily timeline rather than stored as deadline-driven list items.

**Live prototype:** https://rhead04.github.io/TimeWise/

## Key features
- Non-linear timeline (busy hours expand, empty hours compress)
- Mandatory time estimation when adding a task
- Live overrun detection with conflict warnings
- End-of-day comparison of planned vs. actual time
